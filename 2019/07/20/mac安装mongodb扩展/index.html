mac安装mongodb扩展

Author: glong Date: July 20, 2019  17:40:53 Category: mongodb

问题描述？

项目需要mongodb扩展支持，因而需要安装mongodb扩展

安装途径

  • 通过brew ,通过brew search mongodb,可以找到不同php版本的mongodb,此方法经过谷歌发现在php7版本后就不再支持，因而放弃。
  • 通过pecl, pecl install mongodb

    PECL 是通过 PEAR 打包系统来的 PHP 扩展库仓库。PEAR是可重用PHP组件的框架和分发系统。

前进的道路总是坎坷的，遇到问题，不能退缩，要有誓死解决的心态。

安装流程

编译安装mongodb
  1. 下载扩展包

    wget http://pecl.php.net/get/mongodb-1.4.3.tgz

  2. 解压安装包

    tar -zxvf mongodb-1.4.3.tgz

  3. 进入解压目录，运行phpize

    cd mongodb-1.4.3
    phpize //也可以加上路径，/etc/local/bin/phpize

  4. 安装编译mongodb扩展

    ./configure –with-php-config=/usr/localbin/php-config
    make && make install

  5. 编辑php.ini

    extension= mongodb

    pecl安装mongodb
    见证真正的力量吧。

    pecl install mongodb
